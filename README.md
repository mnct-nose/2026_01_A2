# 2026_01_A2：舞鶴市の天気と気候変動調査

気候変動（地球温暖化）に対する関心を深めるため、2025年における京都府舞鶴市の年間の気温変化と降水量の関係を調査・考察したWebレポートプロジェクトです。
気象庁のオープンデータをPythonで可視化し、SDGs（持続可能な開発目標）の視点を取り入れたカラフルなデザインのWebサイトとして構築しています。

---

## プロジェクト概要

昨今注目されている地球温暖化や気候変動を身近な地域データから捉え直すため、気象庁のデータを用いて以下の項目について分析・可視化を行いました。

* **2025年 舞鶴市の気温の推移**（8月の猛暑や5〜6月の激しい寒暖差の分析）
* **2025年 舞鶴市の降水量の推移**（梅雨時期や秋冬の降水傾向の分析）
* **気温と降水量の相関関係の考察**（降水が観測された日における気温の低下傾向の検証）

---

## ファイル構成

本プロジェクトのディレクトリ構成および各ファイルの役割は以下の通りです。

```text
2026_01_A2/
├── index.html                      # レポートを表示するメインのHTMLファイル（SDGsデザイン仕様）
├── README.md                       # 本ファイル（プロジェクトの説明書）
├── analysisreport.md               # データの分析内容をまとめたマークダウンレポート
├── conflict.html                   # 競合確認用、またはバックアップ用のHTMLファイル
│
├── [気温データ関連]
│   ├── tenki(temp).csv             # 気象庁からダウンロードした2025年舞鶴市の気温データ
│   ├── tenki(temp).py              # 気温データをグラフ化するためのPythonスクリプト
│   └── tenki(temp).png             # 【図1】2025年舞鶴市の気温の推移グラフ
│
├── [降水量データ関連]
│   ├── tenki(precipitation).csv    # 気象庁からダウンロードした2025年舞鶴市の降水量データ
│   ├── tenki(precipitation).py     # 降水量データをグラフ化するためのPythonスクリプト
│   └── tenki(precipitation).png    # 【図2】2025年舞鶴市の降水の推移グラフ
│
└── [相関分析データ関連]
    ├── tenki(temp_precipitation).py # 気温と降水量を複合分析するためのPythonスクリプト
    ├── tenki(temp_precipitation).png# 気温と降水量を重ね合わせた複合グラフ
    └── temp&precipitation.png.png  # 【図3】気温・降水の相関推移グラフ（低下日を赤マーク）
```
